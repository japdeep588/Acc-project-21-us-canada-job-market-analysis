# 🇺🇸🇨🇦 US & Canada Data Analyst Job Market Analysis

**Tools:** Excel · Power BI · Python (Pandas) | **Type:** Independent Research Project | **Scope:** 48,000+ job postings

---

## 📌 Project Overview

This project analyses **48,000+ Data Analyst job postings** from the US and Canadian job markets to uncover hiring trends, in-demand skills, and salary benchmarks — built to support data-driven career and curriculum decisions at **Analytics Career Connect (ACC)**.

> **Key Finding:** A **$24,600 salary gap** exists between US and Canadian Data Analyst markets, with SQL, Python, and Power BI consistently ranking as top demanded skills across both regions.

---

## 🎯 Business Questions Answered

- Which skills are most in-demand for Data Analyst roles in the US vs Canada?
- What is the average salary benchmark across both markets?
- Which cities and industries hire the most Data Analysts?
- How does demand differ between entry-level and senior roles?

## 🔍 Methodology

1. **Data Collection** — Raw datasets sourced from Kaggle (US & Canada job postings)
2. **Data Cleaning** — Python (Pandas) used to handle nulls, standardise formats, and extract skills
3. **Data Modelling** — Cleaned CSVs loaded into Power BI with relationships defined
4. **Analysis & Visualisation** — Interactive Power BI dashboard built to answer business questions
5. **Insight Delivery** — Key findings presented to ACC project stakeholders

---

## 💡 Key Insights

- **SQL** is the #1 demanded skill in both markets
- **US salaries** average ~$24,600 higher than Canadian equivalents
- **Entry-level roles** disproportionately require Power BI and Excel over Python
- Top hiring cities: **New York, Toronto, Chicago, Vancouver**
